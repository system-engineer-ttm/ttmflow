# TTMFlow

ระบบจัดการคำขอภายในองค์กรของ Talk to Me Co., Ltd. — ISO 9001:2015 form management built on Next.js.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## What's inside

- Dashboard, Flows (with builder), form picker, form filler, approve/sign, requests list, notifications log, integrations, template builder
- Bilingual TH/EN, 5 role views (Requester / Approver / IT / Admin / Auditor), light + dark, accent themes — all switchable from the bottom-right Tweaks panel
- Mock data only; nothing persists across reload
